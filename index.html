El proyecto tiene como objetivo la realización del automatismo necesario para el control del riego. La idea es realizar un control eficiente de la forma más asequible posible, utilizando una amplia variedad de materiales e incluso re-utilizándolos. De esta manera seria un sistema que podría comprarlo cualquier persona, con poco presupuesto. Para saber cuándo efectuar el riego, se pensó en medir la humedad del terreno mediante unos sensores existentes para Arduino.
